このブログは Next.js の静的サイト生成(SSG)機能を使って、GitHub Pages 上でホストしています。

このページのように `posts/` フォルダに Markdown ファイルを追加すると、ビルド時にHTMLへ変換され、サイトに恒久的に反映されます。

反映までの流れは次の通りです。

1. `posts/*.md` に記事を追加してコミット・プッシュする
2. GitHub Actions が自動でビルド(`next build`)を実行する
3. ビルド済みの静的ファイルが GitHub Pages に自動デプロイされる

トップページの「かんたん投稿(下書き)」フォームから投稿した記事は、この仕組みとは別に、あなたのブラウザの `localStorage` に保存され、その場で一覧に表示されます。ただしそれは **あなたのブラウザだけ** に見える下書きで、他の人がこのサイトを開いても表示されません。本当に全員に公開するには、上記の1〜3の手順で Markdown ファイルとして取り込む必要があります。
